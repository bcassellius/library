### Description
Lovely Librarian is a responsive PWA that is used by librarians for internal library tasks. It is a MERN SPA that follows mobile-first practices.

### Technologies:
- React
- JavaScript
- JSX
- CSS
- MongoDB
- Node.JS
- GraphQL
